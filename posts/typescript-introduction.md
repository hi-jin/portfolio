TypeScript는 자바스크립트의 상위 집합 언어로, 정적 타입을 지원하여 더 안정적인 코드 작성을 가능하게 합니다.

## TypeScript의 장점

1. 정적 타입 검사
2. 더 나은 IDE 지원
3. 객체 지향 프로그래밍 기능 강화
4. 모듈 시스템 지원

## 기본 문법 소개

- 변수와 함수에 타입 지정하기
- 인터페이스 사용하기
- 클래스와 상속
- 제네릭 프로그래밍
