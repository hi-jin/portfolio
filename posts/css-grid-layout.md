CSS Grid Layout은 웹 페이지의 레이아웃을 쉽고 효율적으로 구성할 수 있게 해주는 강력한 도구입니다.

## Grid의 기본 개념

- Grid Container
- Grid Item
- Grid Line
- Grid Track
- Grid Cell
- Grid Area

## 주요 속성

1. display: grid
2. grid-template-columns
3. grid-template-rows
4. gap
5. grid-column
6. grid-row
